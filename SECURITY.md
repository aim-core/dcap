# Security Policy

## Reporting a Vulnerability

DCAP takes security seriously. If you discover a vulnerability, please email:
**security@dcap.dev**

We will respond within 48 hours.

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.4.x   | ✅ |
| < 0.4   | ❌ |

## Security Architecture

DCAP is deterministic. No runtime code execution. No network calls during analysis.
